# 🐟 大连海洋大学官网公开信息采集器

> 一个**只用 Python 标准库**写的轻量爬虫，把学校官网分散的新闻、通知和公开附件链接整理成本地可读文件。无需登录、零第三方依赖、并发采集 + 自动重试。

## ✨ 特性

- 📚 **纯标准库**：只用到 `urllib` / `html.parser` / `concurrent.futures`，`pip install` 都不用
- ⚡ **并发采集**：多线程拉取正文（默认 4 线程，可一键调大）
- 🔁 **自动重试**：超时 / 网络抖动自动退避重试，学校官网再慢也不易中断
- 📝 **整洁输出**：生成 3 个带 BOM 的 UTF-8 文本（Windows 记事本直接看中文）+ 1 份 JSON
- 🖱️ **双击即用**：附带 `一键启动采集器.bat`，Windows 上完全不用敲命令
- 🧪 **有测试**：内置 `unittest`，改完代码一行命令自测
- 🔒 **只采公开内容**：不登录、不绕认证、遵守官网域名白名单

## 📦 安装

1. 安装 **Python 3.10 或更高版本**，安装时务必勾选 **Add Python to PATH**。
2. 下载本项目（克隆或下载 ZIP 解压）到任意普通文件夹，例如 `E:\DLOU爬虫`。
3. 完成，无需其他依赖。

> 如果双击启动后窗口提示 `Python was not found`，说明没装 Python 或没加进 PATH，回到第 1 步重装即可。

## 🚀 使用

### 方式一：双击启动（推荐新手）

1. 在项目文件夹里双击 **`一键启动采集器.bat`**。
2. 稍等片刻，程序会采集学校官网公开信息（默认尽量快地采集）。
3. 完成后自动打开 `output` 文件夹。
4. 阅读顺序：**`采集报告.txt`**（看概览）→ **`正文合集.txt`**（逐篇读）→ **`文章清单.txt`**（只看目录）。

默认只采集每个栏目第 1 页、最多 10 篇，适合第一次试用。

### 方式二：命令行（想多采一点）

在项目文件夹空白处按住 **Shift + 右键** → “在终端中打开”，然后：

```powershell
# 每个栏目翻 3 页，最多 60 篇
python -m dlou_crawler --pages 3 --max-articles 60
```

常用参数：

| 参数 | 含义 | 建议 |
| --- | --- | --- |
| `--pages 3` | 每个栏目最多翻几页列表 | 初学者建议 ≤ 3 |
| `--max-articles 60` | 最多读取多少篇文章正文 | 按需调整 |
| `--concurrency 8` | 并发拉取正文的线程数（默认 4） | 网站慢可调大 |
| `--delay 1` | 两次请求间隔秒数（默认 0，即不限速） | 稳妥起见可设 1 左右 |
| `--output 路径` | 结果保存位置 | 如 `--output D:\我的学校通知` |
| `--download-files` | 顺便下载文章里的公开附件 | 文件可能较大，谨慎使用 |

下载附件示例：

```powershell
python -m dlou_crawler --pages 1 --max-articles 20 --download-files
```

## 📄 输出说明

每次运行的结果保存在 `output/` 文件夹：

| 文件 | 推荐用途 | 内容 |
| --- | --- | --- |
| `文章清单.txt` | 快速浏览 | 所有文章的标题、栏目、日期、原文链接一览 |
| `正文合集.txt` | 逐篇阅读 | 每篇文章完整正文（含元信息），用分隔线区分 |
| `采集报告.txt` | 了解概览 | 采集统计、文件说明与警告记录 |
| `articles.json` | 二次开发 | 最完整的结构化数据 |
| `files/` | 仅用 `--download-files` 时出现 | 下载成功的公开附件 |

> 再次运行会覆盖同一 `output/` 里的结果。想保留某次结果，先把整个 `output/` 复制或改名，例如 `output-2026-07-19`。

## ⚙️ 工作原理

1. 按预设栏目（学校主页新闻 / 信息公告 / 新闻网）广度优先扫描列表页，收集文章链接。
2. 用线程池并发拉取每篇文章的正文、发布日期与附件链接；请求失败自动重试。
3. 按标题去重后，写出上面四份结果文件。

全程只访问学校官网的公开页面，遇到需要登录的页面会保留标题与原文链接并提示未采集，绝不尝试绕过限制。

## ⚠️ 使用边界

- 仅采集网站公开内容，供个人学习与阅读使用。
- 大规模采集时请适度控制频率（用 `--delay` 设间隔），不要频繁重复运行以免给服务器造成压力。
- 不要将采集内容用于商业转载或批量传播。
- 不要修改程序以绕过登录、验证码或访问限制。

## ❓ 常见问题

**双击后窗口一闪而过？** 启动文件最后会暂停。若仍一闪而过，右键 `.bat` → 编辑，确认内容没被其他软件改动；也可在终端里跑上面的命令行看报错。

**为什么有的文章没有正文 / 附件？** 常见原因是网页本身要求统一身份认证、附件非公开，或网站临时无法访问。程序不绕这些限制，请通过“原文链接”在学校网站正常查看。

**用记事本打开是乱码？** 结果文件均为带 BOM 的 UTF-8，Windows 记事本可正常显示中文。若用更老的软件仍乱码，请改用记事本或 VS Code 并以 UTF-8 打开。

**运行很慢？** 速度由 `--delay` 控制（默认 0，即尽量快）。学校网站响应慢时可加 `--delay 1`；大规模采集时适度降频，避免给服务器压力。
