# DLOUWebsiteCrawler

> 大连海洋大学官网公开信息采集器 —— 一键采集全校 30+ 栏目，离线浏览，搜索即得。

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://www.python.org/)
[![Release](https://img.shields.io/badge/Release-v4.0.0-blue)](https://github.com/yuexuan2025/DLOUWebsiteCrawler/releases)

## 功能特点

- **全并发采集** — 所有栏目同时扫描，几分钟搞定全校新闻
- **智能分类** — 按学校官网结构自动分为 4 大类：学校要闻 / 通知公告 / 学生发展 / 各学院动态
- **内置阅读器** — 三栏式浏览（栏目 → 文章列表 → 正文），不用开浏览器
- **HTML 报告** — 一键在浏览器中打开精美报告，支持搜索、排序、折叠
- **一键保存** — 结果可导出到任意位置，U盘拷走离线看
- **单文件 EXE** — 双击即用，无需安装 Python，无需配置环境
- **安全可靠** — 仅访问学校官网域名，不收集用户数据，不登录、不绕过限制

## 快速开始

### 方式一：EXE 直接运行（推荐）

1. 下载最新版 `DLOUWebsiteCrawler.exe`
2. 双击打开
3. 点击「开始采集」
4. 等待采集完成，在下方浏览结果

### 方式二：源码运行

```bash
git clone https://github.com/yuexuan2025/DLOUWebsiteCrawler.git
cd DLOUWebsiteCrawler
pip install -e .
python run.py
```

## 输出文件

采集结果保存在 `output` 文件夹（可自定义路径）：

| 文件 | 说明 |
|------|------|
| `采集报告.html` | 精美 HTML 报告，支持搜索、排序、折叠 |
| `文章清单.txt` | 按栏目分组的文章目录 |
| `正文合集.txt` | 每篇文章的完整正文 |
| `articles.json` | 结构化数据 |
| `files/` | 文章附件（勾选下载时生成） |

## 技术栈

- **纯 Python 标准库爬虫** — `urllib` + `html.parser`，零第三方依赖
- **Tkinter GUI** — 系统原生界面，启动快、体积小
- **ThreadPoolExecutor** — 全并发采集，效率拉满
- **PyInstaller** — 单文件 EXE 打包

## 安全声明

- 仅采集学校官网 **公开内容**
- 不登录、不绕过任何访问限制
- 域名白名单机制，只访问 `*.dlou.edu.cn`
- 不收集、不上传任何用户数据
- 仅供学习和研究使用
