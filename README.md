# DLOUWebsiteCrawler

> 大连海洋大学官网公开信息采集器 —— 一键采集全校 30+ 栏目，离线浏览，搜索即得。

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Release](https://img.shields.io/badge/Release-v3.0.0-blue)](https://github.com/yuexuan2025/DLOUWebsiteCrawler/releases)

---

## ✨ 功能特点

- **🚀 全并发采集** — 所有栏目同时扫描，20 线程全速读取，几分钟搞定全校新闻
- **📊 智能分类** — 按学校官网结构自动分为 4 大类：学校要闻 / 通知公告 / 学生发展 / 各学院动态
- **🌐 内置 HTML 报告** — 程序内直接看精美报告，支持搜索、排序、折叠，不用开浏览器
- **📄 三栏式浏览** — 栏目导航 → 文章列表 → 正文预览，高效阅读
- **💾 一键保存** — 结果可导出到任意位置，U盘拷走离线看
- **🖥️ 单文件 EXE** — 双击即用，无需安装 Python，无需配置环境
- **🔒 安全可靠** — 仅访问学校官网域名，不收集用户数据，不登录、不绕过限制

## 📦 快速开始

### 方式一：EXE 直接运行（推荐）

1. 下载最新版 `DLOUWebsiteCrawler.exe`
2. 双击打开
3. 点击「开始采集」
4. 等待采集完成，在下方浏览结果

### 方式二：源码运行

```bash
git clone https://github.com/yuexuan2025/DLOUWebsiteCrawler.git
cd DLOUWebsiteCrawler
pip install -e .
python run.py
```

## 📁 输出文件

采集结果保存在 `output` 文件夹（可自定义路径）：

| 文件 | 说明 |
|------|------|
| `采集报告.html` | 精美 HTML 报告，支持搜索、排序、折叠 |
| `文章清单.txt` | 按栏目分组的文章目录 |
| `正文合集.txt` | 每篇文章的完整正文 |
| `articles.json` | 结构化数据（开发者用） |
| `files/` | 文章附件（勾选下载时生成） |

## 🛠️ 技术栈

- **纯 Python 标准库爬虫** — `urllib` + `html.parser`，无第三方依赖也能跑
- **Tkinter GUI** — 系统原生界面，启动快、体积小
- **tkhtmlview** — 内置 HTML 渲染，程序内直接看报告
- **ThreadPoolExecutor** — 全并发采集，效率拉满
- **PyInstaller** — 单文件 EXE 打包，双击即用

## 📝 开发说明

### 项目结构

```
DLOUWebsiteCrawler/
├── dlou_crawler/        # 核心代码
│   ├── crawler.py       # 采集器主逻辑
│   ├── gui.py           # 图形界面
│   ├── client.py        # HTTP 客户端（线程安全）
│   ├── html_tools.py    # HTML 解析工具
│   ├── output.py        # 报告生成
│   ├── models.py        # 数据模型
│   └── cli.py           # 命令行模式
├── tests/               # 单元测试
├── run.py               # 程序入口
├── build.spec           # PyInstaller 打包配置
└── pyproject.toml       # 项目配置
```

### 打包 EXE

```bash
pip install pyinstaller tkhtmlview pillow
pyinstaller build.spec --noconfirm
```

生成的 EXE 在 `dist/DLOUWebsiteCrawler.exe`。

### 运行测试

```bash
python -m unittest discover tests
```
