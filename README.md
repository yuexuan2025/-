# DLOUWebsiteCrawler

大连海洋大学官网公开信息采集器 —— 一键采集学校要闻、通知公告、招生就业、各学院动态，生成离线报告。

## 快速使用

1. **下载并运行**：下载 `DLOUWebsiteCrawler.exe`，双击打开
2. **点击「开始采集」**：程序自动以最优配置全速采集
3. **浏览结果**：采集完成后，页面下方自动显示结果
   - **文章浏览**：按「学校要闻 / 通知公告 / 学生发展 / 各学院动态」四大类分组，三栏式快速浏览
   - **HTML 报告**：内置完整 HTML 报告，支持搜索、排序、折叠
4. **保存结果**：点击「保存结果」可导出到任意文件夹，离线也能看

## 采集范围

| 分类 | 包含栏目 |
|------|---------|
| 学校要闻 | 学校要闻、综合新闻、校园快讯、媒体报道、校园喜报 |
| 通知公告 | 信息公告、学术海大、今日活动、下载专区 |
| 学生发展 | 本科生教育、研究生教育、本科生招生、研究生招生、本科生就业、研究生就业、继续教育 |
| 各学院动态 | 水产与生命学院、海洋科技与环境学院、食品科学与工程学院、海洋与土木工程学院、机械与动力工程学院、航海与船舶工程学院、信息工程学院、经济管理学院、海洋法律与人文学院、外国语学院、中新合作学院、马克思主义学院、体育与教育学院、应用技术学院 |

## 功能特点

- 🚀 **全并发采集**：所有栏目同时扫描，20 线程全速读取
- 📊 **智能分类**：按学校官网结构自动分类，方便查找
- 🌐 **内置 HTML 报告**：无需浏览器，直接在程序内查看精美报告
- 📄 **三栏式浏览**：栏目 → 文章列表 → 正文预览，高效阅读
- 💾 **一键保存**：结果可导出到任意位置，离线可用
- 🔒 **安全可靠**：仅访问学校官网域名，不收集用户数据

## 输出文件

采集结果保存在 `output` 文件夹（或自定义路径）中：

- **采集报告.html** — 浏览器打开，支持搜索、排序、折叠
- **文章清单.txt** — 按栏目分组的文章目录
- **正文合集.txt** — 每篇文章的完整正文
- **articles.json** — 结构化数据（开发者用）
- **files/** — 文章附件（勾选"下载文章附件"时生成）

## 从源码运行

```bash
pip install -e .
python run.py
```

## 打包 EXE

```bash
pip install pyinstaller
pyinstaller build.spec --noconfirm
```

生成的 EXE 在 `dist/DLOUWebsiteCrawler.exe`。

## 安全声明

- 仅采集学校官网公开内容，不登录、不绕过任何限制
- 只访问 `*.dlou.edu.cn` 域名下的页面
- 不收集、不上传任何用户数据
- 仅供学习和研究使用
